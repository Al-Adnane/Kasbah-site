How it works

Three things happen every time you hit Send.

1

Intercept

The browser extension catches the Send button click on ChatGPT or Claude before the message leaves your browser.

2

Scan

Your message is scanned locally for API keys, passwords, tokens, connection strings, and 20+ secret patterns. Risk score: 0-100.

3

Decide

You see exactly what was found and choose: Block or Allow. Nothing leaves your machine without your explicit approval.

What it catches

Real patterns. Detected instantly. Before the model sees them.

🔑

API Keys

OpenAI, AWS, GitHub, Slack tokens detected by pattern matching.

sk-proj-4f8a9b2c7d...
🔒

Passwords & Secrets

Database passwords, environment variables, auth tokens.

🗂

Connection Strings

MongoDB, PostgreSQL, MySQL, Redis URIs with credentials.

Private Keys

Setup in 3 steps

1

Open the DMG, drag Kasbah Guard to Applications, launch it. The guard starts automatically on port 8788.

2

In Opera: go to opera://extensions, enable Developer mode, click "Load unpacked" and select the extension folder. (Same flow for Chrome/Brave/Edge.)

3

Open ChatGPT or Claude. Type something with a fake secret. Hit Send. Kasbah will intercept it and show you the checkpoint. That's it.
