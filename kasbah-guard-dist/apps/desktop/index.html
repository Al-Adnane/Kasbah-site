Attack: Replay the ticket

Intercepted a ticket? Try using it twice. Kasbah tracks every consumed ticket. Second attempt = instant denial.

// Use ticket once — works
await execute(ticket);
// Try again — BLOCKED
await execute(ticket);
Run safe flow first...

Attack: Tool swap

Got a ticket for db.query? Try using it for shell.exec. Tickets are cryptographically bound.

// Ticket for database
const t = await kasbah.decide({ tool: 'db.query' });
// Privilege escalation attempt
await kasbah.consume(t, { tool: 'shell.exec' });
